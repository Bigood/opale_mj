Module

Un Module Opale Starter est constitué de Divisions qui correspondent aux différentes parties du document.

Division

La Division se compose de :

  • Grains de contenu permettant d'exposer et de structurer l'information ;

  • Activités pédagogiques interactives (exercices), à savoir :

    • Question à choix unique (QCU),

    • Question à choix multiple (QCM).

Le type d'information exposée dans le cours est explicité par des balises pédagogiques adaptées (Information, Attention, Rappel, Fondamental, Exemple,...).

À l'intérieur d'un grain, l'information peut être regroupée en autant de Parties titrées que vous le souhaitez.

Ressources

Opale Starter vous offre la possibilité d'insérer de nombreuses ressources multimédia dans vos contenus.

Par ressources, on entend :

  • les images ( png, jpg, gif),
  • les vidéos ( mp4, WebM,...) et leurs fichiers de sous-titrage ( srt, WebVTT),
  • les ressources audio ( mp3, Ogg, opus),
  • les Animations Flash,
  • les Tableaux ods,
  • les Graphiques odg,
  • les formules mathématiques ( Équations odf),
  • les documents bureautiques courants ( Word, Excel, PowerPoint, Texte odt, Présentation odp, pdf, zip).
